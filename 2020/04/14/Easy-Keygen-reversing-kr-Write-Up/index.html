分析

在IDA中观察反汇编指令

main函数中发现了 “Input Name: “

  • stosb / stosw / stosd 这三条指令是将 al / ax / eax 寄存器的值传送到 ** [edi] ** 中, 前面加上 rep 指令时则重复执行ecx次, 同时指定edi+1或edi-1. 在这里的作用只是初始化栈空间.
  • repne scasb 这是两条指令, rep n e也就是 repeat not equal 不等于时重复执行 scasb, scasb执行cx次[al] - [di]每次比较di自动变化. repne scasb在一起就是每次比较[al]和[di]中的数据, 相等则停止.

4010A3~4010B1的操作相当于strlen获取字符串长度.
大概了解到这是一个for循环, ebp为字串的下标, esi为三个char变量的下标, 每次循环将ebp指向的字符与esi指向的字符进行异或操作, 之后esi和ebp加一, 直到ebp大于等于字符串时跳出循环, 当esi大于等于3时清零.
